## Packages
framer-motion | Smooth animations for page transitions and interactions
@solana/wallet-adapter-react | React hooks for Solana wallet adapter
@solana/wallet-adapter-react-ui | UI components for wallet connection
@solana/wallet-adapter-wallets | Wallet adapters (Phantom, etc.)
@solana/web3.js | Solana web3 library
lucide-react | Beautiful icons
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge Tailwind classes

## Notes
- Using @solana/wallet-adapter-react for wallet connection (mocked visual connection if no wallet present)
- The app simulates on-chain data storage using the local backend API defined in shared/routes.ts
- IPFS uploads are simulated via the backend for now
- Styling uses a "Campus Blue" theme with modern, clean typography
